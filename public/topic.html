Chosen Topic: Web Accessibility

Accessibility is about designing websites usable by everyone, including people with disabilities. This page explains key ideas and practical examples.

Why it matters

Accessible sites improve usability, inclusivity, and overall quality of the user experience.

Good practices

  • Use semantic HTML (headings, landmarks)
  • Provide labels for form controls
  • Write meaningful link text
  • Add alt text for images

Conclusion

Small decisions in HTML and CSS can significantly improve accessibility for many users.
